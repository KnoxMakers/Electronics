Little Blinky - Blink the LED on the Arduino Board



Install the Arduino IDE

Plug a USB cable into your computer and into the Arduino Nano. The USB cable is probably one of these, if you don't have one.

In the Arduino IDE, set up the hardware:

  • Tools->Board->Arduino Nano
  • Tools->Processor->Atmega328P (Old Bootloader)
  • Tools->Port-> whatever port appears in this list when you plug in the Arduino


Replace everything in the IDE code window with the following:


void setup() { // set the LED pin as an output
  pinMode(LED_BUILTIN,OUTPUT);
}

// the loop function runs over and over forever, after setup() runs
void loop() {
  digitalWrite(LED_BUILTIN,HIGH); // turn the LED on
  delay(250); // wait for 1/4 second
  digitalWrite(LED_BUILTIN, LOW); // turn the LED off
  delay(250); // wait for 1/4 second
}


Now click on the check mark in the upper left of the IDE to compile the code. It should complete without errors.

Now upload it to your Arduino by clicking on theright arrow symbol in the upper left area of the IDE.
