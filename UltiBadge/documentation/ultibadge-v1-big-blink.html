Big Blinky - Scroll Text on the LED Matrices



If you haven't successfully programmed your Arduino yet, go back to the blink sketch and make it happen.

In the Arduino IDE, install two libraries:

  • Tools->Manage Libraries - Fine MD_MAX72XX and click install
  • Tools->Manage Libraries - Fine MD_Parola and click install
In the Arduino IDE, set up the hardware:
  • Tools->Board->MD_MAX72XX
  • Tools->Processor->Atmega328P (Old Bootloader)
  • Tools->Port-> whatever port appears in this list when you plug in the Arduino

Download ScrollingText.ino from the KnoxMakers Github, load that into the IDE, compile and upload it.
