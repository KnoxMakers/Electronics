UltiBadge V1

Getting Started Guide



What you've got to play with:

  • Arduino Nano
  • Accelerometer (and possibly Gyro)
  • 2 LED displays, 8x8 matrix
  • 5 tactile switches, a d-pad and an action switch
  • Infrared transmit and receive
  • Code examples contributed by your friends at KM


Here's what all we've got to offer in terms of documentation:

  • All about the hardware. Parts list, how to build it, board modifications
  • Little Blink (Blink the LED on the Arduino board)
  • Scrolling Text: Scrolls some text across the LCD Matrices
  • Switch Test: Displays the number of the switch pressed on an LED matrix and sends a notification via the serial port
  • Bouncing Ball (bounces a ball around the screen)
  • Tetris - Download, install from GitHub
